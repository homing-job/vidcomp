## アプリ概要
サーキットのラップタイム更新を目的にした動画比較サービス<br>
スマホでの使用を想定<br>
<br>
<br>

## 作った経緯
コンマ数秒を意識して動画を比較するサービスがなかった。
サーキットを走ってる人はコンマ数秒を気にするので超重要。
走ってる人はyoutube上に車載動画を上げている人が多い。
自分との比較にも使えるし、他人との比較にも使えて便利なのではと思い作りました。
